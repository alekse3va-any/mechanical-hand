# Алгоритм работы

## Основной цикл

1. Пользователь открывает локальный сайт.
2. Сайт запрашивает текущий статус у Flask-сервера.
3. Flask-сервер отправляет Arduino команду `STATUS`.
4. Arduino считывает кнопку в ладони и возвращает JSON.
5. Сайт отображает состояние руки и наличие предмета.

## Команда “Сжать руку”

1. Пользователь нажимает кнопку “Сжать руку”.
2. Сайт отправляет POST-запрос `/api/close`.
3. Flask-сервер отправляет Arduino команду `CLOSE`.
4. Arduino поворачивает сервопривод в положение закрытия.
5. Сервопривод натягивает общую тягу.
6. Пальцы сгибаются.
7. Arduino считывает кнопку в ладони.
8. Сайт получает ответ и обновляет статус.

## Команда “Разжать руку”

1. Пользователь нажимает кнопку “Разжать руку”.
2. Сайт отправляет POST-запрос `/api/open`.
3. Flask-сервер отправляет Arduino команду `OPEN`.
4. Arduino поворачивает сервопривод в положение раскрытия.
5. Тяга ослабляется.
6. Пружинки возвращают пальцы в раскрытое положение.
7. Сайт получает ответ и обновляет статус.

## Формат ответа Arduino

```json
{
  "ok": true,
  "event": "closed",
  "hand": "closed",
  "object": "yes",
  "sensor": "pressed",
  "message": "ok"
}
```
