# Управляемая механическая рука

Проект представляет собой механическую руку с веб-интерфейсом управления. Пользователь отправляет команду через сайт, Flask-сервер передаёт её на Arduino по USB, Arduino управляет сервоприводом и возвращает состояние датчика в ладони.

## Идея проекта

Рука имеет четыре пальца. Каждый палец состоит из фаланг и сгибается за счёт нити, работающей как сухожилие. Нити всех пальцев объединены в одну общую тягу. Тяга крепится к катушке сервопривода. При вращении сервопривода тяга натягивается, пальцы сгибаются, и рука переходит в состояние захвата. Разгибание выполняется возвратными пружинами.

В ладони установлена кнопка. Если предмет нажимает на кнопку, система определяет, что объект находится в руке.

## 3D-модель

Основой механической части выбрана готовая 3D-печатная модель **InMoov Hand**. Она используется как база для ладони, пальцев, шарниров и сухожильной передачи.

Ссылки на модели:

- https://inmoov.fr/inmoov-hand/
- https://inmoov.fr/hand-and-forarm/
- https://www.thingiverse.com/thing:17773

Модель используется в учебных некоммерческих целях с указанием автора и источника.

## Состояния системы

| Состояние | Описание |
|---|---|
| open | рука разжата |
| closed | рука сжата |
| object: yes | объект обнаружен кнопкой в ладони |
| object: no | объект не обнаружен |

## Запуск

1. Загрузить скетч из папки `arduino` в Arduino.
2. Подключить сервопривод к пину D9.
3. Подключить кнопку ладони к пину D2 и GND.
4. Подключить внешний источник питания сервопривода.
5. Соединить GND Arduino и GND источника питания сервопривода.
6. Установить зависимости и запустить сервер:

```bash
pip install -r requirements.txt
python app.py
```

После запуска открыть в браузере:

```text
http://127.0.0.1:5000
```

## Управление

На сайте есть три кнопки:

- «Разжать руку»;
- «Сжать руку»;
- «Обновить статус».

Кнопки отправляют запросы на Flask-сервер. Сервер передаёт команды Arduino:

```text
OPEN
CLOSE
STATUS
```

Arduino управляет сервоприводом и возвращает состояние руки и датчика ладони.
