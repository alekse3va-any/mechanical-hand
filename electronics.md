# Электронная часть

## Компоненты

| Компонент | Назначение |
|---|---|
| Arduino Uno или совместимая плата | Выполняет команды от сайта |
| MG996R или аналогичный сервопривод | Натягивает общую тягу пальцев |
| Кнопка/концевик | Определяет наличие предмета в ладони |
| Внешний источник 5–6 В | Питает сервопривод |
| Провода | Соединяют элементы схемы |
| USB-кабель | Соединяет Arduino с компьютером |

## Подключение

| Элемент | Контакт | Подключение |
|---|---|---|
| Сервопривод | Signal | D9 Arduino |
| Сервопривод | VCC | +5–6 В внешнего источника |
| Сервопривод | GND | GND внешнего источника и GND Arduino |
| Кнопка | контакт 1 | D2 Arduino |
| Кнопка | контакт 2 | GND Arduino |

Кнопка подключается с использованием режима `INPUT_PULLUP`. Поэтому отдельный резистор для кнопки не нужен: в ненажатом состоянии вход имеет уровень HIGH, а при нажатии замыкается на GND и становится LOW.

## Питание сервопривода

Сервопривод нельзя питать от пина 5V Arduino. Под нагрузкой он потребляет больше тока, чем может стабильно дать плата. Поэтому сервопривод питается от отдельного источника 5–6 В. Земля Arduino и земля источника питания должны быть соединены.

## Настройка углов

В файле `arduino/mechanical_hand_controller.ino` заданы углы:

```cpp
const int OPEN_ANGLE = 25;
const int CLOSED_ANGLE = 115;
```

После сборки их нужно подстроить:

- если рука не раскрывается полностью, изменить `OPEN_ANGLE`;
- если пальцы перетягиваются, уменьшить `CLOSED_ANGLE`;
- если пальцы закрываются слабо, немного увеличить `CLOSED_ANGLE`.

Настройку нужно выполнять без предмета в руке и без сильного натяжения нитей.
